# Monalisa Tech Solutions – Premium Property Investment Platform

A luxury real estate landing page designed to connect investors, property buyers, and developers through a modern digital experience.

This project showcases premium property developments, investment opportunities, escrow transparency, flexible payment plans, and investor-focused services in a clean, responsive, and visually engaging interface.

---

## Preview

The platform includes:

- Premium property listings
- Investment insights and market statistics
- Golden Visa information
- Virtual property experiences
- Payment plan calculator
- Escrow transparency tracking
- Developer showcase
- Investor testimonials
- Consultation booking forms
- WhatsApp contact integration

---

## Built With

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts (Cormorant & Inter)

No frameworks or external dependencies are required.

---

## Features

### Property Discovery
Explore curated real estate developments with detailed investment information.

### Investment Insights
Present market trends, rental yields, and portfolio performance metrics.

### Golden Visa Information
Guide investors through residency-by-investment opportunities.

### Escrow Transparency
Display project milestones and payment progress clearly.

### Payment Planning
Interactive payment calculator for investment planning.

### Responsive Design
Optimized for desktop, tablet, and mobile devices.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/MONALISA2003-CMD/monalisa-property-platform.git
```

Navigate into the project folder:

```bash
cd monalisa-property-platform
```

Open the project in your browser:

```bash
index.html
```

or simply double-click the file.
